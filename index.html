Machine Learning Intern

During my internship, I spearheaded the development of an anomaly detection model aimed at identifying and addressing faults in complex time-series datasets. The project centered around leveraging Temporal Convolutional Networks (TCNs), a cutting-edge deep learning architecture tailored for sequential data. These networks excel at capturing temporal dependencies and identifying subtle patterns that signal anomalies, making them an ideal choice for the industrial processes we aimed to monitor. My responsibilities encompassed not only the technical implementation of the model but also collaborative efforts with domain experts to ensure practical applicability and reliability.

SurveyMage

SurveyMage is a full-stack web application that allows users to create, share, and analyze surveys. The platform provides a seamless user experience with secure authentication using Firebase and a robust backend powered by Node.js, Express.js, and MongoDB. SurveyMage also offers a comprehensive data analytics feature, enabling survey creators to visualize and export response data effortlessly.

PlantGuard

PlantGuard is a Flask-based web application designed to identify plant diseases from images using a pre-trained deep learning model. It employs the ResNet-18 architecture, fine-tuned on the Plant Pathology dataset, to classify multiple plant diseases effectively.

ConvoLink is a cutting-edge Android application designed to provide seamless and interactive video conferencing capabilities. Built using Flutter and powered by the ZEGOCLOUD API, ConvoLink offers a comprehensive suite of features to enable high-quality video communication. With user authentication powered by Firebase and real-time call tracking, the application is tailored to meet both personal and professional needs.
